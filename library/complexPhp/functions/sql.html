sql():

sql(query, onDone, onError, conn) function is used to run mysql query.

Example:

 setDefaultConn("localhost", "root", "", "test");

 sql("UPDATE entries SET name='John Doe' WHERE id='100'", 
					function()
					{
						
						echo "Record Updated Successfully.";
					}, 
					function($error){

						echo "Failed to update record. Error: ".$error;
					}
				);

Output

 Record Updated Successfully

Example 2

Without setDefaultConn() it will throw error.

// setDefaultConn("localhost", "root", "", "test");

 sql("UPDATE entries SET name='John Doe' WHERE id='100'", 
					function()
					{
						
						echo "Record Updated Successfully.";
					}, 
					function($error){

						echo "Failed to update record. Error: ".$error;
					}
				);

Output

// Error ComplexPHP: Database name is null, Use setDefaultConn() or newSqli()

To use sql() without setDefaultConn() , just pass your new connection to sql().

 $myconn = new mysqli("localhost", "root", "", "test");

 sql("UPDATE entries SET name='John Doe' WHERE id='100'", 
					function()
					{
						
						echo "Record Updated Successfully.";
					}, 
					function($error){

						echo "Failed to update record. Error: ".$error;
					},
					$myconn
				);

Output:

// Record Updated Successfully.
