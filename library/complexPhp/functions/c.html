c():

c(string) function is called complex class function. It returns string as encripted varable.

Example:

<div class="[c: myClass]"></div>
<div class="[c: pretty]"></div>
<div class="[c: myClass]"></div>

Output

 <div class="gbz32vsah"></div>
 <div class="ablwzGqtwsY"></div>
 <div class="gbz32vsah"></div>
